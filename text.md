# text

## 預設字體

{% hint style="info" %}
雖有預設字體，為確保xml會有字體註解，請逐一個別選定字體。
{% endhint %}

> _這樣xml 才有註解字體_

## 靜態文字

{% hint style="info" %}
先在none狀態下調整字體大小，最後再以「自動收縮」處理
{% endhint %}

> _只有在"無"與"自動收縮"情況下，才不會變更到文字物件寬度。_
