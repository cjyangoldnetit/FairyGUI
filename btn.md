##### TA應用

| 按鈕 | 規格 |
| :---: | :--- |
| 系統按鈕\(選廳頁\) | 不勾「選中」 /不連接場景 /設為普通按鈕 \(由前端設定動作\) |
| 系統按鈕\_左上\(主遊戲\) | 90X90 /距離邊界20 |
| 操作按鈕\(主遊戲\) | 180X90 |
| 結算畫面 | 僅「繼續遊戲/Continue」 |
| 金額刷新按鈕\(全遊戲\) | 皆無需重新整理按鈕，可製作並隱藏 |
